*1.Інструкцію запуску:*

1)Cтворення проекту командою: npx create-expo-app@latest --template;
2)Встановлення бібліотеки для роботи з файлами командою: npx expo install expo-file-system;

3)Написання коду;

4)Запуск сервера командою: npx expo start;

5)Сканування QR-коду програмою Expo GO через телефон;

*2.Опис реалізованого функціоналу:*

Навігація між папками:

Система дозволяє здійснювати перехід у вкладені директорії шляхом натискання на відповідний об'єкт у списку. 

Передбачено механізм повернення до батьківської директорії.

Створення об'єктів:

Застосунок підтримує створення нових папок та текстових файлів формату .txt. 

Користувач має можливість задати назву нового об'єкта, а для файлів — також вказати початковий текстовий вміст через відповідні поля введення.

Перегляд та редагування вмісту файлів:

Реалізовано функцію відкриття текстових документів для зчитування даних. 

При виборі файлу відкривається вбудований редактор, який дозволяє не лише переглядати вміст, а й модифікувати його з подальшим збереженням змін у файловій системі.

Видалення файлів і папок:

Передбачено можливість повного видалення об'єктів. Функція викликається натисканням на відповідний елемент керування у списку файлів.

Перегляд детальної інформації:

Для кожного об'єкта файлової системи реалізовано виведення розширених атрибутів. 

Користувач може переглянути точну назву, розмір об'єкта в кілобайтах, а також дату та час останньої модифікації файлу чи папки.

Перегляд статистики пам’яті пристрою:

На головному екрані застосунку відображається актуальна інформація про стан пам'яті пристрою. 

Виводиться загальний обсяг дискового простору та обсяг вільної пам'яті, доступної для використання. 

Передбачено функцію примусового оновлення статистики для відстеження змін після операцій запису або видалення.
