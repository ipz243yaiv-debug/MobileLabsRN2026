Цей додаток є базовою розробкою на React Native з використанням інструментарію Expo. У застосунку реалізовано кастомну шапку з логотипом Житомирської політехніки, систему навігації між екранами: Новини, Галерея, Профіль та футер. В кожній вкладці є певна інформація.
Інструкція із запуску: 
1) Клонування репозиторію на комп'ютер
2) Встановлення необхідних бібліотек
3) Запуск серверу командою npx expo start.
4) Щоб відкрити на емуляторі достатньо натиснути клавішу "a", а на звичаному смартфоні потрібно відсканувати QR-код через програму Expo Go.

Способи запуску мобільного додатка: 
1. Запуск через Expo Go
Цей спосіб призначений для швидкого перегляду додатку на реальному смартфоні. Його головна особливість у тому, що не потрібно встановлювати програмне забезпечення типу Android Studio. Це найкращий варіант для швидкої перевірки інтерфейсу та жестів.

2. Запуск через Android Emulator 
Використовується для тестування у віртуальному середовищі безпосередньо на комп'ютері. Основна відмінність полягає у можливості перевірити роботу додатка на різних моделях телефонів та різних версіях Android. 

3. Запуск на фізичному пристрої через USB
Використовується для фінального тестування продуктивності. На відміну від емулятора, цей метод дає найточніше уявлення про те, як додаток споживає пам'ять та наскільки швидко працює на реальному пристрої.
